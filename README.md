# scotland-yard
COM5043 assessment 1

## Connect and use database:

- git clone <repo>
- cd <repo>
- docker compose up -d
- cd server
- npm install
- copy .env.example .env
- npm run dev

# Model
* Docker → creates the database
* Prisma → creates the tables
* Node → reads/writes game data
* Adminer → lets you view the DB

# Access
* http://localhost:8080
